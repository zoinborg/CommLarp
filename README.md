# CommLarp
Commbank Larp Script (Overriders Balances) Total credits + net position = sum of both accounts.
Usage Tested on Chrome, Firefox, LibreWolf, Edge. Recommended Firefox + Violentmonkey.

Install Violentmonkey https://violentmonkey.github.io/get-it/ or Tampermonkey (for chrome and edge) https://www.tampermonkey.net/index.php 
Download usescript and click create new script in violent or tampermonkey then paste in and save as. 
Press P to open GUI. Total credits + net position = sum of both accounts.
